11 This, What?, Who?

Let’s see how to say the sentence “This is a man.”

The word for “this” is هَـٰذَا. It is pronounced haadhaa.

In Arabic we usually don’t write any word for “is”. So then we can say:

هَـٰذَا رَجُل
“This is a man.”

Let’s see now how to say “This is a girl.” When we are talking about a girl, or a noun that has ة we don’t use هَـٰذَا. Instead we use هَـٰذِهِ haadhihi.

هَـٰذِهِ بِنْت
“This is a girl.”

هَـٰذِهِ قِطَّة
“This is a cat.”

هَـٰذَا كَلْب
“This is a dog.”

Questions

We can ask a question “What is this?” The word for “What” is مَا. Again, we don’t write any word for “is”.

So if we are asking about noun which does not have ة then we say:

مَا هَـٰذَا؟ هَـٰذَا كَلْب
“What is this? This is a dog.”

But if we are asking about a noun which has ة then we say:

مَا هَـٰذِهِ؟ هَـٰذِهِ شَجَرَة
“What is this? This is a tree.”

Now if we are talking about a person, then we don’t say “What is this?” Instead we say, “Who is this?”

The word for “who” is مَنْ.

مَنْ هَـٰذَا؟ هَـٰذَا رَجُل
“Who is this? This is a man.”

مَنْ هَـٰذِهِ؟ هَـٰذِهِ بِنْت
“Who is this? This is a girl.”
