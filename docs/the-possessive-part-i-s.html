17 The possessive (Part I: ’s)

In English we can say

  • “a boy’s book”
  • “the woman’s house”
  • “Zayd’s school”

In Arabic we simply place the nouns together but we reverse the order of the two nouns. And only the second noun gets ال for “the”. Here are the Arabic for the above examples:

Arabic English
كِتَاب-وَلَد “a boy’s book”
بَيْت-المَرْأَة “the woman’s house”
مَدْرَسَة-زَيْد “Zayd’s school”

For now we will put a hyphen “-” between the words which are connected with “’s”.

NOTE TO INSTRUCTOR

We’re using a hyphen because two nouns can occur next to each other without being a possessive, e.g., رَأَتْ أُمّ وَلَد which, without case endings can be either

  • “A woman saw a boy.”, or
  • “She saw a boy’s mother.”

So without introducing case endings we think it will be easier to facilitate recognition of the possessive, at least where there is ambiguity.

Also, we we say مَدْرَسَة-زَيْد we will pronounce it madrasat zayd not madrasah zayd.

So when translating from Arabic to English, if you see two nouns together with a hyphen “-” between them, you should translate them with (’s).

Start with last noun. if it has ال then write “the”. Then go backwards and connect nouns with (’s).

سَيَّارَة-الرُّجُل
“the man’s car”

سَيَّارَة-رَجُل
“a man’s car”

سَيَّارَة زَيْد
“Zayd’s car”

By the way we can add more nouns connected with “’s”:

مِفْتَاح بَاب بَيْت رَجُل
“a man’s house’s door’s key”

مِفْتَاح بَاب بَيْت الرَّجُل
“the man’s house’s door’s key”

Note that ال is added only to the last noun.

NOTE TO INSTRUCTOR

It may be more idiomatic in English to say “The key of the door of the house of the man”. But it will be hard to explain to the student how the multiple “the’s” in English are translated.

And in the next chapter, for simplicity, we will reserve “of” for quantities and materials.
